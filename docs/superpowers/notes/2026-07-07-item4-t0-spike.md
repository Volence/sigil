## Evaluator env + ambient-injection choke point
`Evaluator<'a>` (`eval/mod.rs:72`) holds `consts/enums/fns/structs/bitfields/newtypes/datas/offsets:
HashMap<&'a str, &'a ast::‚Ä¶Decl>`, populated by `index_items` (`eval/mod.rs:263`) via
`with_file` (`eval/mod.rs:252`). No existing ambient/prelude seam. `with_file` is called from
`layout.rs` (`eval_data_with_root`, `eval_offsets_with_root`) and proc lowering ‚Äî these are the sites to
thread `&Ambient`. Undefined label refs do NOT error in eval (pass through as `Expr::Sym`, fail at link)
‚Äî so cross-module LABEL refs "just work" as fixups; only comptime TYPE/const refs need ambient injection.
